## Dashboard (Power BI)

This folder contains static images from the Power BI dashboard created for the
Customer Churn Prediction project.

### Visualizations Included:
- **churn_overview.png** – Overall distribution of churn vs non-churn customers
- **churn_by_gender.png** – Churn comparison across gender categories
- **charges_vs_tenure.png** – Customer distribution based on monthly charges and tenure

Note: A static version is shared due to account and publishing limitations.
